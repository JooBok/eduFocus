Apache Kafka 로컬에서 실행
---------------------------------------------------
> 리눅스 설치가 우선되어야 함.</br>
> Windows에서 cmd와 Powershell을 관리자 권한으로 실행.</br>
>` wsl --install Ubuntu-22.04` 입력 -> 본 프로젝트에서는 특정 버전을 사용.</br>
